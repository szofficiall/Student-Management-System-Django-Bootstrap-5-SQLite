# 📌 About Project

This project is a complete Student Management System developed using Django Framework.

The system allows administrators to manage students, departments, subjects, marks, authentication, and report cards efficiently through a clean and responsive web interface.

The application is designed for academies, schools, and educational institutes.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/student-management-system.git
```

Move into project

```bash
cd student-management-system
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Linux / Mac

```bash
source venv/bin/activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Migrations

```bash
python manage.py makemigrations

python manage.py migrate
```

Run Server

```bash
python manage.py runserver
```

Open

```
http://127.0.0.1:8000/
```
